### Nodejs Version:

```v16.14.0```

### Before activate the website you need to install:

1. please go to <b>client</b> dir and put this instruction:

     ```npm install``` 

2. please go to <b>app</b> dir and put this instruction:

     ```npm install``` 

### How to activate the website

1. please go to <b>client</b> dir and put this instruction:

     ```npm start``` 

2. please go to <b>app</b> dir and put this instruction:

     ```npm start```
